銀(Ag)の化学的性質

1. 基本情報

  • 元素記号: Ag (原子番号 47)
  • 特徴: 金属光沢が最も強く、電気伝導性・熱伝導性が最大。

2. 銀の製法

  • 焙焼: (1) 2Ag₂S + 3O₂ → 2Ag₂O + 2SO₂
  • 熱分解: (2) 2Ag₂O → 4Ag + O₂

3. 銀の化学反応

  • 希硝酸: (6) 3Ag + 4HNO₃(希) → 3AgNO₃ + NO↑ + 2H₂O
  • 濃硝酸: (7) Ag + 2HNO₃(濃) → AgNO₃ + NO₂↑ + H₂O

4. 銀の化合物と沈殿

  • 塩化物イオン: (25) Ag⁺ + Cl⁻ → AgCl↓
  • クロム酸イオン: (33) 2Ag⁺ + CrO₄²⁻ → Ag₂CrO₄↓

5. 銀鏡反応

  • 反応式: (35) RCHO + 2[Ag(NH₃)₂]⁺ + 2OH⁻ → RCOO⁻ + 2Ag↓ + 4NH₃ + H₂O
