銀(Ag)の化学的性質

基本情報

  • 元素記号: Ag (ラテン語 Argentum より)
  • 原子番号: 47
  • 電子配置: [Kr]4d¹⁰5s¹
  • 酸化数: +1(銀イオン Ag⁺)が主
  • 常温での状態: 銀白色の金属(金属光沢が最も強い)

銀の製法

※ 各項目をクリックすると反応式が表示されます

1. 乾式製錬法

  • (1) 硫化銀の焙焼: 2Ag₂S + 3O₂ → 2Ag₂O + 2SO₂
  • (2) 酸化銀の熱分解: 2Ag₂O → 4Ag + O₂

2. 青化法(シアン化法)

  • (4) 銀の溶解: 4Ag + 8NaCN + O₂ + 2H₂O → 4Na[Ag(CN)₂] + 4NaOH
  • (5) 亜鉛による還元: 2[Ag(CN)₂]⁻ + Zn → [Zn(CN)₄]²⁻ + 2Ag

銀イオンの沈殿反応

  • (25) Ag⁺ + Cl⁻: AgCl↓ (白色沈殿)
  • (26) Ag⁺ + Br⁻: AgBr↓ (淡黄色沈殿)
  • (27) Ag⁺ + I⁻: AgI↓ (黄色沈殿)

銀鏡反応

アルデヒドの還元性を利用した重要な検出反応です。

  • (35) 反応式: RCHO + 2[Ag(NH₃)₂]⁺ + 2OH⁻ → RCOO⁻ + 2Ag↓ + 4NH₃ + H₂O
