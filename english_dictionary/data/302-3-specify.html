# 302-3

specify 動詞

主な意味
〜を明確に述べる、指定する、詳述する
ニュアンス: 具体的に「これ」と指し示したり、詳細な条件を挙げたりすることを指します。
例文
Please specify which color you want. どの色が欲しいか指定してください。
The contract specifies that the work must be finished by June. 契約書には、仕事は6月までに完了しなければならないと明記（指定）されている。
語源 specific + -ify
類義語 (Synonyms)
detail(詳述する)
stipulate(規定する)
対義語 (Antonyms)
generalize(一般化する)
関連語
specific(形容詞：具体的な)
