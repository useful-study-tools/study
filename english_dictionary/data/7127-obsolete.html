# 7127

obsolete 形容詞

主な意味
すたれた、時代遅れの
ニュアンス: 新しい技術や方法の登場によって、古くなり使われなくなった状態を指します。
例文
Many old technologies have become obsolete. 多くの古い技術は時代遅れになった。
The factory uses obsolete machinery. その工場は時代遅れの機械を使っている。
This software is now obsolete and no longer supported. このソフトウェアは現在では時代遅れで、もはやサポートされていない。
The typewriter is largely obsolete today. タイプライターは今日ではほとんど時代遅れである。
The company's business model became obsolete. その会社のビジネスモデルは時代遅れになった。
語源 ラテン語 'obsoletus'（古びた、使い古した）に由来。
