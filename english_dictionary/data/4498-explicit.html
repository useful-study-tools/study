# 4498

explicit 形容詞

主な意味
明白な、明確な、あからさまな
ニュアンス: 包み隠さず、はっきりと述べられている状態を指します。反対に「性的・暴力的な描写が露骨な」という意味でも使われます。
例文
He gave me explicit instructions on how to get there. 彼はそこへの行き方について明確な指示をくれた。
The film contains explicit scenes of violence. その映画には露骨な暴力シーンが含まれている。
語源 ex (外に) + plicare (折る) → 折りたたまれていない、開かれた
類義語 (Synonyms)
clear(はっきりした)
definite(明確な)
overt(公然の)
対義語 (Antonyms)
implicit(暗黙の)
vague(曖昧な)
ambiguous(多義的な)
関連語
explicate(動詞：詳しく説明する)
explicitly(副詞：はっきりと)
