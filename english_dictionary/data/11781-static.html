# 11781

static 形容詞

主な意味
静的な、変化のない、活気のない
ニュアンス: 動かず一定のままであること。物理学的な「静止」や、社会・経済の「停滞」を指します。
例文
Oil prices have remained static for several months. 石油価格は数ヶ月間、変化のない（横ばいの）ままだ。
The population of the village is static. その村の人口は増減がない（停滞している）。
語源 histastai（立たせる）
類義語 (Synonyms)
stationary(静止した)
unchanging(変わらない)
fixed(固定された)
対義語 (Antonyms)
dynamic(動的な)
changing(変化する)
active(活動的な)
関連語
statistics(名詞：統計)
