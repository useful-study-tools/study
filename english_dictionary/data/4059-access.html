# 4059

access 名詞、動詞

主な意味
【名】利用する権利、接近、入手方法 【動】～にアクセスする、～を利用する
ニュアンス: 場所へ近づくことや、情報・施設などを利用できる「可能性・権利」を指します。
名詞の例文
Students have free access to the library's online database. 学生は図書館のオンラインデータベースを自由に利用する権利がある。
The only access to the village is by boat. その村への唯一の接近手段は船だ。
動詞の例文
You need a password to access the file. そのファイルにアクセスする（を利用する）にはパスワードが必要だ。
語源 ad (の方へ) + cedere (行く)
類義語 (Synonyms)
entry(入り口、入場)
approach(接近)
admission(入場、入会許可)
対義語 (Antonyms)
exclusion(除外、排除)
denial(拒否)
関連語
accessible(形容詞：近づきやすい、利用可能な)
