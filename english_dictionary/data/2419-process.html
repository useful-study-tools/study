# 2419

process 名詞・動詞

主な意味
過程、工程、プロセス；処理する
ニュアンス: ある結果に至るための「一連の手順・段階」を指します。動詞では情報・食品・申請書などを「処理する」という意味で使われます。'in the process of〜'（〜の途中で）も頻出表現です。
例文（名詞：過程・プロセス）
Learning a language is a long and gradual process. 言語を習得することは長くて段階的なプロセスだ。
The application process takes about two weeks. 申請の手続きには約2週間かかる。
例文（動詞：処理する）
The system can process thousands of requests per second. そのシステムは1秒間に数千件のリクエストを処理できる。
