# 506-2 (関連語)

pirate 名詞/動詞

意味
海賊、著作権侵害者 / 〜を違法コピーする
ニュアンス: 海上で略奪を行う者、または著作物を違法に複製・配布する者を指します。動詞では違法にコピーすることを表します。
例文（名詞）
Pirates attacked the merchant ship. 海賊が商船を襲撃した。
Software pirates face legal consequences. ソフトウェアの違法コピー者は法的な結果に直面する。
例文（動詞）
It is illegal to pirate movies. 映画を違法コピーすることは違法だ。
語源 ギリシャ語 peirates (襲撃者)
→ peiran (試みる) から
類義語 (Synonyms)
buccaneer(海賊)
corsair(海賊)
bootlegger(密売人)
対義語 (Antonyms)
関連語 piracy (名詞：海賊行為)
