# 5586

analyze 動詞

主な意味
～を分析する
ニュアンス: 「複雑なものを細かい部分に分解して調べる」という意味です。データや問題の構造を明らかにする時に使います。
例文
The data was analyzed by experts. そのデータは専門家によって分析された。
We need to analyze the problem before solving it. 問題を解決する前に分析する必要がある。
The scientist analyzed the chemical composition. その科学者は化学組成を分析した。
Let's analyze the results carefully. 結果を注意深く分析しよう。
The software helps analyze customer behavior. そのソフトウェアは顧客行動の分析を助ける。
語源 ギリシャ語 analyein（分解する）に由来します。
