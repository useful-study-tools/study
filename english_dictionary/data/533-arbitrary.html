# 533

arbitrary 形容詞

主な意味
恣意的な、独断的な、任意の
ニュアンス: 理由や原則に基づかず、個人の気まぐれや独断で決められることを表します。しばしば否定的なニュアンスを持ち、不公平さを示唆します。
例文
The decision seemed arbitrary and unfair. その決定は恣意的で不公平に思えた。
The rules appear arbitrary with no clear logic. その規則は明確な論理がなく恣意的に見える。
He made an arbitrary choice without consulting others. 彼は他人に相談せず独断的な選択をした。
語源 ラテン語 arbitrarius (裁定者の)
→ arbiter (仲裁人) から
類義語 (Synonyms)
random(無作為の)
capricious(気まぐれな)
whimsical(気まぐれな)
unreasonable(不合理な)
対義語 (Antonyms)
reasoned(理にかなった)
logical(論理的な)
systematic(体系的な)
関連語
arbitrarily(副詞：恣意的に)
