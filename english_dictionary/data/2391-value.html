# 2391

value 名詞・動詞

主な意味
価値、値打ち、価格；大切にする、評価する
ニュアンス: 名詞としては経済的・道徳的・感情的な「価値・重要性」を表します。複数形 values は「価値観・倫理観」の意味になります。動詞では「価値を認める・大切にする」という意味で使われます。
例文（名詞：価値）
The painting has increased in value over the years. その絵画は年々価値が上がっている。
Traditional values are still respected in many communities. 多くのコミュニティでは伝統的な価値観がいまだに尊重されている。
例文（動詞：大切にする）
I truly value your advice. あなたのアドバイスを本当に大切にしています。
語源 ラテン語 valere（強い・価値がある）← 古フランス語 valoir
