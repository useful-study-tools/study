# 7210

clone 名詞・動詞

主な意味
クローン、複製、～のクローンを作る
ニュアンス: 遺伝的に同一の個体や、まったく同じコピーを作ることを指す。
例文
Dolly the sheep was the first mammal to be cloned. 羊のドリーは、クローン化された最初の哺乳類だった。
The movie is just a clone of an earlier successful film. その映画は以前の成功作の単なる模倣（クローン）だ。
Scientists are debating the ethics of human cloning. 科学者たちはヒトクローンの倫理について議論している。
It is possible to clone certain plants from a single leaf. 1枚の葉から特定の植物のクローンを作ることが可能だ。
He felt like a clone, dressed exactly like everyone else. 彼は皆とまったく同じ格好をして、まるでクローンのように感じた。
語源 ギリシャ語 klon（小枝）から。挿し木で増やすことに由来。
