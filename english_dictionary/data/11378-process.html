# 11378

process 名詞、他動詞

主な意味
①過程 [他] ②～を加工する，処理する
ニュアンス: 一連の手順や進展を指す名詞と、データや食品などを扱う動詞。
意味①：過程（名詞）
Learning a language is a long process. 言語を学ぶのは長い過程である。
意味②：加工・処理する（動詞）
The data is being processed by the computer. データはコンピュータで処理されている。
We process cheese in this factory. この工場ではチーズを加工している。
語源 pro (前へ) + cedere (行く) → 前へ進むこと
類義語 (Synonyms)
procedure(手順)
handle(扱う)
refine(精製する)
対義語 (Antonyms)
関連語
processor(名詞：プロセッサ)
procession(名詞：行列)
