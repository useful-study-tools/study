# 7165

flaw 名詞

主な意味
ひび、欠陥
ニュアンス: 完全な状態を損なう小さな傷や欠点を指します。製品の欠陥や性格の短所などに使われます。
例文
There is a flaw in the diamond. そのダイヤモンドには傷がある。
The argument has a serious logical flaw. その議論には深刻な論理的な欠陥がある。
The software was recalled due to a security flaw. そのソフトウェアはセキュリティ上の欠陥のためにリコールされた。
His only flaw is that he is sometimes too generous. 彼の唯一の欠点は、時々寛大すぎることだ。
The engineer discovered a flaw in the design. その技術者は設計の欠陥を発見した。
語源 古ノルド語 'flaga'（石板の破片）に由来。
