# 2212

fix 動詞

主な意味
修理する、固定する、決定する
ニュアンス: 壊れたものを直す（修理する）こと、動かないように留める（固定する）こと、あるいは日時などを確定させる（決定する）ことを意味します。
例文（修理する・直す）
Can you fix my broken watch? 私の壊れた時計を修理してくれますか？
He is trying to fix the computer. 彼はコンピューターを直そうとしている。
例文（固定する・決定する）
Fix the poster to the wall. ポスターを壁に固定しなさい。
We need to fix a date for the meeting. 会議の日程を決定する必要がある。
語源 ラテン語 figere（固定する）から
類義語 (Synonyms)
repair(修理する)
attach東進上級英単語1000 attach鉄壁 (取り付ける)
set(決める)
対義語 (Antonyms)
break(壊す)
damage(ダメージを与える)
change(変える（決定するの対として）)
関連語
