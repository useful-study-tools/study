# 4819

intricate 形容詞

主な意味
複雑な、込み入った
ニュアンス: 細かい部分がたくさんあり、理解したり作ったりするのが難しいほど「精巧に複雑」な様子を指します。
例文
The watch has an intricate mechanism. その時計は複雑な（精巧な）仕組みを持っている。
She designed an intricate pattern for the carpet. 彼女はカーペットのために込み入った模様をデザインした。
語源 in (中に) + tricae (困り事、障害) → もつれた
類義語 (Synonyms)
complex(複合的な)
complicated(複雑な)
elaborate(手の込んだ)
対義語 (Antonyms)
simple(単純な)
plain(簡素な)
straightforward(わかりやすい)
関連語
intricacy(名詞：複雑さ)
