# 11540

relevant 形容詞

主な意味
関連のある、適切な
ニュアンス: 単に関係があるだけでなく、その場において「重要で意味がある」ことを指します。
例文
Please include only the information relevant to this case. この件に関連のある情報だけを含めてください。
語源 re (再び) + levare (持ち上げる) → 助けになる、関連する
類義語 (Synonyms)
related(関係のある)
pertinent(適切な、的を射た)
applicable(適用できる)
対義語 (Antonyms)
irrelevant(無関係な)
unrelated(関連のない)
関連語
relevance(名詞：関連性、重要性)
