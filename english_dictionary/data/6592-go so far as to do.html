# 6592

go so far as to do 動詞句

主な意味
～までする
ニュアンス: 「極端な行動を取る」という意味。
例文
He went so far as to accuse her of lying. 彼は彼女を嘘つきだと非難するまでになった。
She went so far as to quit her job. 彼女は仕事を辞めるまでになった。
I wouldn't go so far as to say he's a genius. 彼が天才だと言うまではしないだろう。
He went so far as to threaten violence. 彼は暴力をほのめかすまでになった。
They went so far as to hire a private investigator. 彼らは私立探偵を雇うまでになった。
語源 go（行く）+ so far（そこまで）+ as to do（～するまで）に由来。
類義語 (Synonyms)
go to the extreme of(極端なことをする)
take the step of(～する手段を取る)
resort to(～に訴える)
対義語 (Antonyms)
stop short of(～する直前で止める)
avoid (避ける)
refrain from(控える)
関連語
go(行く)
