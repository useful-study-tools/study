# 5074

core 名詞

主な意味
中心, 核心； (果物の)芯
ニュアンス: 「最も重要な部分」「本質」を表します。リンゴの「芯」から転じて、問題の核心や会社のコア事業などに使われます。
例文
The core of the problem is lack of communication. 問題の核心はコミュニケーション不足です。
He ate the apple, leaving only the core. 彼はリンゴを食べて、芯だけ残した。
These values are at the core of our company. これらの価値観は私たちの会社の中核にある。
We need to focus on our core business. 私たちは中核事業に集中する必要がある。
The Earth's core is extremely hot. 地球の核は非常に熱い。
語源 ラテン語 cor（心臓）に由来します。
