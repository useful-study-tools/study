# 4210

consensus 名詞

主な意味
（意見の）一致、合意、コンセンサス
ニュアンス: グループ内での総意、あるいは大多数が納得している状態を指します。
例文
The committee reached a consensus on the new policy. 委員会は新方針について合意に達した。
There is a general consensus that global warming is a serious threat. 地球温暖化が深刻な脅威であるという一般的な共通認識がある。
語源 ラテン語 con（共に）+ sentire（感じる）→ 共に感じる
類義語 (Synonyms)
agreement(合意)
accord (一致)
unanimity(満場一致)
対義語 (Antonyms)
disagreement(不一致)
dissent(異議)
