# 7208

icy wasteland 名詞句

主な意味
氷に覆われた不毛の地
ニュアンス: 厳しい寒さで生命がほとんど存在しない地域を描写する。
例文
Antarctica is a vast icy wasteland. 南極は広大な氷の不毛の地である。
The explorers crossed the icy wasteland for months. 探検家たちは何ヶ月もかけて氷の不毛の地を横断した。
Nothing grows in that icy wasteland. その氷の不毛の地では何も育たない。
They were lost in an icy wasteland with no shelter. 彼らは避難所のない氷の荒地で迷子になった。
The movie is set in a post-apocalyptic icy wasteland. その映画は終末後の氷の荒地が舞台だ。
語源 icy（氷の）+ wasteland（荒地）
類義語 (Synonyms)
frozen desert(凍った砂漠)
polar desert(極地砂漠)
glacier plain(氷河平原)
対義語 (Antonyms)
lush forest(豊かな森)
fertile land(肥沃な土地)
tropical paradise(熱帯の楽園)
関連語
