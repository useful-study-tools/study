# 370-3

confidential 形容詞

主な意味
機密の、秘密の、親任の
ニュアンス: 外部に漏らしてはいけない公的な機密事項や、プライベートな秘密を指します。
例文
These documents contain highly confidential information. これらの書類には極めて機密性の高い情報が含まれている。
He works as a confidential advisor to the president. 彼は大統領の親任の（腹心の）顧問として働いている。
語源 confidence + ial
類義語 (Synonyms)
secret(秘密の)
private(私的な)
classified(機密扱いの)
対義語 (Antonyms)
public(公の)
open(開かれた)
関連語
confidentiality(名詞：守秘義務)
