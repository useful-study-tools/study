# 11454

digest 他動詞、名詞

主な意味
①～を消化する ②～を理解する [名] ③要約
ニュアンス: 食べ物や情報を細かく砕いて取り込みやすい形にすること。名詞では情報のまとめを指します。
意味①：消化する
It takes about two hours to digest a heavy meal. 重い食事を消化するのに約2時間かかる。
意味②：理解する
I need some time to digest all this information. このすべての情報を消化（理解）するのに少し時間が必要だ。
意味③：要約（名詞）
He read a digest of the report. 彼はその報告書の要約を読んだ。
語源 di (離れて) + gerere (運ぶ) → 分けて運ぶ、整理する
類義語 (Synonyms)
absorb(吸収する)
understand(理解する)
summary(要約)
対義語 (Antonyms)
vomit(吐き出す)
misunderstand(誤解する)
expansion(拡張)
関連語
digestion(名詞：消化)
digestive(形容詞：消化の)
