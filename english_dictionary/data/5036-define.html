# 5036

define 動詞

主な意味
～を定義する, 明確にする
ニュアンス: 言葉や概念の「意味を明確に定める」ことです。また、範囲や境界を「はっきりさせる」という意味でも使われます。
例文
Can you define this word for me? この単語を定義してもらえますか？
The dictionary defines 'happiness' as a state of well-being. 辞書は『幸福』を満足の状態と定義している。
It's difficult to define exactly what makes a good leader. 何が良いリーダーを構成するかを正確に定義するのは難しい。
The boundaries of the property are clearly defined. その土地の境界は明確に定められている。
Your role in this project is not well defined. このプロジェクトにおけるあなたの役割は明確に定義されていない。
語源 ラテン語 definire（境界を定める）から来ています。
