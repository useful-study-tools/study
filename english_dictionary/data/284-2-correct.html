# 284-2

correct 形容詞、動詞

主な意味
正しい、適切な、〜を訂正する、添削する
ニュアンス: 事実に基づいた「正しさ」と、間違いを「直す」動作の両方を指します。
例文（形容詞）
Please make sure all the information is correct. すべての情報が正しいか確認してください。
例文（動詞）
The teacher corrected the students' essays. 教師は生徒の作文を添削（修正）した。
語源 ラテン語 con（完全に）+ regere（導く、支配する）
類義語 (Synonyms)
accurate(正確な)
right(正しい)
amend(修正する)
対義語 (Antonyms)
incorrect(間違った)
wrong(誤った)
関連語
correctly(副詞：正しく)
