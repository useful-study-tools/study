# 6511

make up 動詞句

主な意味
(～を)埋め合せる、取り戻す(for)； (～と)仲直りする； ～(話)をでっち上げる； 化粧する
ニュアンス: 「補う」「仲直りする」「作り話をする」「化粧をする」など多義的な句動詞。
例文
We need to make up for lost time. 失った時間を取り戻す必要がある。
They kissed and made up after the argument. 彼らは喧嘩の後、キスして仲直りした。
He made up a story about being sick. 彼は病気だという話をでっち上げた。
She takes ten minutes to make up every morning. 彼女は毎朝化粧に10分かかる。
Hard work can make up for a lack of talent. 勤勉は才能の不足を補うことができる。
語源 make（作る）+ up（上に）に由来。
