# 11928

drastic 形容詞

主な意味
抜本的な、思い切った、猛烈な
ニュアンス: 状況を大きく変えるような、非常に強力で急激な手段や変化を指します。
例文
We had to take drastic measures to save the company. 会社を救うために、私たちは思い切った（抜本的な）措置をとらなければならなかった。
There has been a drastic change in the weather. 天候に劇的な（猛烈な）変化があった。
語源 dran (行う) → 行動力のある、効き目の強い
