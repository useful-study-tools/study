# 2153

enable 動詞

主な意味
有効にする、可能にする
ニュアンス: 人や物が何かをできるように能力や手段を与えること（可能にする）、またはシステムなどの機能を「有効にする」ことを表します。
例文（可能にする）
The internet enables us to work from home. インターネットは私たちが在宅勤務することを可能にする。
This tool enables easier cutting. この道具はより簡単な切断を可能にする。
例文（有効にする）
Please enable Wi-Fi on your device. 端末のWi-Fiを有効にしてください。
The security system has been enabled. セキュリティシステムが有効にされた。
語源 en-（〜にする）+ able（できる）
