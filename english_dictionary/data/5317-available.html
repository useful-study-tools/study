# 5317

available 形容詞

主な意味
入手できる, 利用できる, 手があいている
ニュアンス: 「使用や取得が可能な状態」を指します。製品の在庫、人の予定、情報の入手可能性などに使います。
例文
Is this seat available? この席は空いていますか？
The new phone is now available in stores. その新しい電話は現在店頭で入手できる。
I'm available for a meeting tomorrow. 明日は会議に参加できます。
The report is available online. その報告書はオンラインで入手可能だ。
There is no available parking space. 利用可能な駐車スペースはない。
語源 古フランス語 availer（役立つ）に由来します。
類義語 (Synonyms)
accessible(アクセス可能な)
obtainable(入手可能な)
free (暇な)
対義語 (Antonyms)
unavailable(利用できない)
occupied(占有された)
unobtainable(入手不可能な)
関連語
availability(名詞：利用可能性)
avail (動詞：役立つ)
