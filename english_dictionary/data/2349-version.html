# 2349

version 名詞

主な意味
版、バージョン、〜版、見解
ニュアンス: ソフトウェアの「バージョン」から、映画や本の「〜版」、ある出来事についての「〜の話・見解」まで幅広く使われます。
例文（版・バージョン）
The latest version of the software was released yesterday. そのソフトウェアの最新バージョンが昨日リリースされた。
Have you seen the film version of the novel? その小説の映画版を見ましたか？
例文（見解・話）
Each witness gave a different version of events. 各目撃者は出来事について異なる話をした。
語源 ラテン語 versio（vertere「回す、向ける、変える」）→ 向きを変えたもの＝別の形
