# 2445

effort 名詞

主な意味
努力、骨折り、尽力
ニュアンス: 目標のために「力や時間を注ぎ込むこと」を指します。'make an effort'（努力する）、'put effort into〜'（〜に努力を注ぐ）、'despite all efforts'（あらゆる努力にもかかわらず）などが頻出表現です。
例文（努力・尽力）
She made every effort to finish the project on time. 彼女はプロジェクトを期限内に終わらせるためにあらゆる努力をした。
It takes a lot of effort to learn a new language. 新しい言語を習得するには多大な努力が必要だ。
Global efforts to reduce emissions must be accelerated. 排出量削減に向けた世界的な取り組みを加速させなければならない。
語源 古フランス語 esfort ← esforcier（力を尽くす）← ex-（外に）+ fortis（強い）
