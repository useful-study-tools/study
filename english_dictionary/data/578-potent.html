# 578

potent 形容詞

主な意味
（薬などが）強力な、効力のある、影響力のある
ニュアンス: 力（power）が内包されており、強い効果や影響を及ぼすことができる状態を指します。特に薬の効き目や、象徴的なシンボルの持つ力、あるいは説得力のある議論などについて使われます。
例文
This is a very potent drug with many side effects. これは非常に強力な（効き目の強い）薬で、多くの副作用がある。
He presented a potent argument for change. 彼は変革に向けた説得力のある（強力な）議論を提示した。
語源 ラテン語 potens (力のある)
→ 可能性がある、力を持っている
類義語 (Synonyms)
powerful(強力な)
effective(効果的な)
influential(影響力のある)
対義語 (Antonyms)
impotent(無力な)
weak(弱い)
ineffective(効果のない)
関連語
potential(名詞/形容詞：可能性/潜在的な)
potency(名詞：効力、権力)
