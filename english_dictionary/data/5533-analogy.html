# 5533

analogy 名詞

主な意味
類推
ニュアンス: 「類似点に基づいて、ある事柄を別の事柄で説明すること」を指します。'draw an analogy between'で「～間に類推を描く」です。
例文
The teacher explained the concept by analogy. 先生は類推によってその概念を説明した。
He drew an analogy between the brain and a computer. 彼は脳とコンピューターの間に類推を描いた。
The analogy is not perfect, but it helps understanding. その類推は完璧ではないが、理解の助けになる。
By analogy, we can guess how the system works. 類推によって、私たちはそのシステムがどう動くか推測できる。
The book uses an analogy of a journey to describe life. その本は人生を描写するために旅の類推を使っている。
