# 521

remedy 名詞/動詞

主な意味
治療法、救済策 / 〜を治す、改善する
ニュアンス: 病気や問題を解決する方法を指します。医学的な治療法だけでなく、社会問題や困難な状況の解決策としても使われます。
例文（名詞）
There is no simple remedy for poverty. 貧困には簡単な解決策がない。
Herbal remedies are becoming more popular. ハーブ療法がますます人気になっている。
例文（動詞）
We must remedy this situation immediately. この状況を直ちに改善しなければならない。
語源 ラテン語 remedium (治療)
→ re (再び) + mederi (癒す)
類義語 (Synonyms)
cure(治療法)
treatment(治療)
solution(解決策)
fix(修正する（動詞）)
対義語 (Antonyms)
disease(病気)
problem(問題)
関連語
remedial(形容詞：治療の・補習の)
