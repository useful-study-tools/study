# 422-2 (関連語)

degradation 名詞

意味
悪化、低下、分解、(品位の)下落
ニュアンス: 質や価値、状態が徐々に損なわれていくプロセスを指します。環境分野では「環境破壊（environmental degradation）」、化学分野では「物質の分解」として頻出します。
例文
Environmental degradation is reaching a critical point in this region. この地域における環境悪化は危機的な状況に達している。
The degradation of traditional values concerns many older citizens. 伝統的価値観の低下が、多くの高齢者の懸念となっている。
語源 degrade (下げる) + ation (こと)
→ 段階を落とすこと
類義語 (Synonyms)
deterioration(悪化)
decline(衰退・低下)
breakdown(崩壊・分解)
対義語 (Antonyms)
improvement(改善)
restoration(修復・復元)
enhancement(向上)
関連語 degrade (動詞：下げる・分解する)
