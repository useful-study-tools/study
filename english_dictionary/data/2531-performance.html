# 2531

performance 名詞

主な意味
遂行、業績、演奏
ニュアンス: 仕事や任務を実行することやその結果（成果・業績）。また、舞台での演技や演奏、機械の性能など幅広い意味を持ちます。
例文（業績、性能）
The company's financial performance was excellent this year. その会社の今年の財務業績は素晴らしかった。
We need to improve the performance of this computer. このコンピュータの性能を向上させる必要がある。
例文（演技、演奏）
Her performance in the movie was outstanding. その映画での彼女の演技は傑出していた。
