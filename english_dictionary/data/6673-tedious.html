# 6673

tedious 形容詞

主な意味
退屈な
ニュアンス: 「長くて退屈な」という意味。
例文
The lecture was so tedious that I fell asleep. その講義はとても退屈で眠ってしまった。
Doing repetitive tasks can be tedious. 繰り返しの作業は退屈になり得る。
The journey was long and tedious. その旅は長くて退屈だった。
He found the book tedious. 彼はその本が退屈だと感じた。
The tedious work made him want to quit. その退屈な仕事は彼に辞めたくなった。
語源 ラテン語 taedium（退屈）に由来。
