# 5502

object 名詞 / 動詞

主な意味
名詞：物体, 対象, 目的； 動詞：(～に)反対する(to～)
ニュアンス: 名詞では「具体的な物」や「目標」。動詞では「反対する」という意味で、'object to'で「～に反対する」です。
例文
What is that object on the table? テーブルの上のあの物体は何ですか？
The object of the game is to score points. そのゲームの目的は得点を取ることです。
I object to your proposal. 私はあなたの提案に反対します。
She is the object of his affection. 彼女は彼の愛情の対象だ。
Many people objected to the new rule. 多くの人がその新しい規則に反対した。
語源 ラテン語 obicere（前に投げる）に由来します。
