# 6083

run for 動詞句

主な意味
～に出馬する
ニュアンス: 「選挙に立候補する」という意味。
例文
She decided to run for president. 彼女は大統領選に出馬することに決めた。
He is running for mayor. 彼は市長選に出馬している。
They encouraged him to run for office. 彼らは彼に公職への立候補を勧めた。
She ran for Congress and won. 彼女は連邦議会に出馬し、勝利した。
He is running for re-election. 彼は再選を目指して出馬している。
語源 run（走る）+ for（～のために）に由来。
