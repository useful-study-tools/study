ベンゼンと芳香族炭化水素

1. 芳香族化合物とは

芳香族化合物は、ベンゼン環に代表される芳香環を持つ化合物の総称である。特徴的な芳香臭を持つものが多いことからこの名がある。

  • 定義: ベンゼン環のような芳香族性(共鳴安定化)を示す環状構造を持つ化合物
  • 脂肪族化合物との違い: 不飽和度が高いが、付加反応より置換反応を起こしやすい
  • 代表例: ベンゼン(C₆H₆)、トルエン(C₆H₅CH₃)、キシレン(C₆H₄(CH₃)₂)

2. ベンゼンの構造

2.1 ケクレ構造

  • 1865年、ケクレが提案した構造: 6員環で単結合と二重結合が交互に並んだ構造
  • しかし、この構造では二重結合の位置が2通り考えられる(共鳴)。

2.2 現代の構造(共鳴理論)

  • ベンゼンは6個の炭素原子が正六角形に配置され、すべてのC-C結合は等価(長さ1.39Å)。
  • これは単結合(C-C 1.54Å)と二重結合(C=C 1.34Å)の中間の長さ。
  • 6個のπ電子が環全体に非局在化(非局在化)している。
  • この安定化を(1) 芳香族性(芳香性)という。
  • 共鳴エネルギー: 約150 kJ/mol(非常に安定)。

2.3 分子軌道

  • 各炭素はsp²混成軌道をとり、平面構造。
  • 残ったp軌道が環の上下で重なり合い、π電子雲を形成。
  • ヒュッケル則: 4n+2個のπ電子を持つ環状共役系は芳香族性を示す(ベンゼンはn=1で6個)。

3. ベンゼンの物理的性質

  • 性状: 無色の液体、特有の芳香臭、沸点80.1℃、融点5.5℃
  • 比重: 0.879(水より軽い)
  • 溶解性: 水に不溶。有機溶媒(エーテル、アルコール、アセトンなど)に溶ける。
  • 毒性: 発がん性(ベンゼンは白血病の原因となる)があるため、取り扱いに注意。

4. ベンゼンの化学反応(置換反応)

ベンゼンは付加反応より置換反応を起こしやすい。これは、付加反応では芳香族性が失われるのに対し、置換反応では芳香族性が保たれるため。

4.1 ニトロ化

  • 濃硝酸と濃硫酸の混酸で処理すると、ニトロベンゼンが生成する。
  • 反応式: (2) C₆H₆ + HNO₃ → C₆H₅NO₂ + H₂O
  • 濃硫酸の役割: ニトロニウムイオン(NO₂⁺)を発生させる。
  • 生成物: ニトロベンゼン(淡黄色の液体、アーモンド臭)。

4.2 スルホン化

  • 濃硫酸または発煙硫酸で処理すると、ベンゼンスルホン酸が生成する。
  • 反応式: (3) C₆H₆ + H₂SO₄ → C₆H₅SO₃H + H₂O
  • 可逆反応で、希硫酸と加熱すると加水分解されてベンゼンに戻る。
  • 生成物: ベンゼンスルホン酸(固体、水に溶ける)。

4.3 ハロゲン化

  • 鉄(III)塩(FeCl₃, FeBr₃)やアルミニウム塩(AlCl₃)などの触媒存在下で、ハロゲンと反応しハロゲン化ベンゼンが生成する。
  • 反応式(塩素): (4) C₆H₆ + Cl₂ → C₆H₅Cl + HCl
  • 反応式(臭素): (5) C₆H₆ + Br₂ → C₆H₅Br + HBr
  • 触媒の役割: ハロゲンを分極させ、求電子種(Cl⁺など)を発生させる。
  • 生成物: クロロベンゼン(液体)、ブロモベンゼン(液体)。

4.4 フリーデル・クラフツ反応

アルキル化

  • ハロゲン化アルキルとAlCl₃などの触媒を用いて、アルキル基を導入する。
  • 反応式: (6) C₆H₆ + R-Cl → C₆H₅-R + HCl
  • 例(トルエン): (7) C₆H₆ + CH₃Cl → C₆H₅CH₃ + HCl
  • 多置換されやすい(さらに反応が進む)欠点がある。

アシル化

  • 酸塩化物(R-COCl)とAlCl₃を用いて、アシル基を導入する。
  • 反応式: (8) C₆H₆ + R-COCl → C₆H₅-CO-R + HCl
  • 例(アセトフェノン): (9) C₆H₆ + CH₃COCl → C₆H₅COCH₃ + HCl

4.5 付加反応(特殊な条件下)

  • 水素添加: ニッケル触媒などで高温高圧下で水素を付加すると、シクロヘキサンになる。
  • 反応式: (10) C₆H₆ + 3H₂ → C₆H₁₂
  • 塩素の付加: 光照射下で塩素を付加すると、BHC(ベンゼンヘキサクロリド, 殺虫剤)が生成する。
  • 反応式: (11) C₆H₆ + 3Cl₂ → C₆H₆Cl₆

5. 置換基の配向性

ベンゼン環にすでに置換基がある場合、次の置換基の導入位置は既存の置換基によって決まる。

5.1 オルト・パラ配向性基

  • オルト位とパラ位を活性化し、そこに置換基が入りやすくなる。
  • 電子供与性基: -OH, -OCH₃, -NH₂, -NHR, -NR₂, -CH₃, -C₂H₅など(アルキル基)
  • 理由: これらの基はベンゼン環に電子を供与し、オルト位とパラ位の電子密度を高める。

5.2 メタ配向性基

  • メタ位を活性化(またはオルト・パラ位を不活性化)し、メタ位に置換基が入りやすくなる。
  • 電子求引性基: -NO₂, -CN, -SO₃H, -CHO, -COR, -COOH, -COOR, -CONH₂など
  • 理由: これらの基はベンゼン環から電子を引き抜き、オルト位とパラ位の電子密度を特に低下させる。

5.3 配向性のまとめ

配向性 置換基の例 特徴
オルト・パラ配向性 -OH, -OCH₃, -NH₂, -CH₃, -C₂H₅, -Cl, -Br, -I(ハロゲンは特殊) 反応を促進(ハロゲンは除く)
メタ配向性 -NO₂, -CN, -SO₃H, -CHO, -COOH, -COOR 反応を抑制

※ハロゲン(-Cl, -Br, -I)はオルト・パラ配向性だが、環を不活性化する(電子求引性)。

6. アルキルベンゼン

6.1 トルエン (C₆H₅CH₃)

  • 性状: 無色の液体、沸点111℃。ベンゼンに似た臭い。
  • 製法: ベンゼンのフリーデル・クラフツアルキル化、または石油の改質。
  • 反応:
    • 側鎖の酸化: 過マンガン酸カリウムなどで酸化すると安息香酸になる。
    • (12) C₆H₅CH₃ + 3[O] → C₆H₅COOH + H₂O
    • ニトロ化: 混酸で処理すると、主にo-ニトロトルエンとp-ニトロトルエンの混合物ができる。
    • スルホン化: 主にo-トルエンスルホン酸とp-トルエンスルホン酸ができる。
  • 用途: 溶剤、安息香酸の原料、TNT(トリニトロトルエン)の原料、ガソリンのオクタン価向上剤。

6.2 キシレン (C₆H₄(CH₃)₂)

  • 3種類の異性体がある。
  • o-キシレン(1,2-ジメチルベンゼン): 沸点144℃
  • m-キシレン(1,3-ジメチルベンゼン): 沸点139℃
  • p-キシレン(1,4-ジメチルベンゼン): 沸点138℃、融点13℃
  • 用途: p-キシレンはテレフタル酸の原料(ポリエステル繊維)。溶剤としても使用。

6.3 クメン (イソプロピルベンゼン, C₆H₅CH(CH₃)₂)

  • 用途: クメン法(フェノールとアセトンの製造)の中間体。

6.4 スチレン (C₆H₅CH=CH₂, ビニルベンゼン)

  • 性状: 無色の液体、沸点145℃。
  • 製法: エチルベンゼンの脱水素。
  • 用途: ポリスチレン(発泡スチロール、プラスチック)の原料。合成ゴム(SBR)の原料。

7. 多環芳香族炭化水素

7.1 ナフタレン (C₁₀H₈)

  • 構造: 2つのベンゼン環が縮合した構造。
  • 性状: 白色の板状結晶、特有の臭い(防虫剤の臭い)、融点80℃、昇華性。
  • 反応: ベンゼンより反応性が高い。ニトロ化やスルホン化は主にα位(1位)に起こる。
  • 用途: 防虫剤(パラジクロロベンゼンに置き換えられつつある)、染料中間体、合成樹脂原料。

7.2 アントラセン (C₁₄H₁₀)

  • 構造: 3つのベンゼン環が直線状に縮合。
  • 性状: 無色の板状結晶、紫外線で青紫色の蛍光を発する。

7.3 フェナントレン (C₁₄H₁₀)

  • 構造: 3つのベンゼン環が角型に縮合。

7.4 発がん性物質

  • ベンゾ[a]ピレンなど、一部の多環芳香族炭化水素は発がん性を持つ。
  • タバコの煙、排気ガス、焼け焦げた食品などに含まれる。

8. ベンゼン・芳香族炭化水素の工業的製法

8.1 コールタールからの分離

  • 石炭の乾留(コークス製造)の際に副生するコールタールを分留して得る。
  • ベンゼン、トルエン、キシレン、ナフタレンなどを含む。

8.2 石油化学からの合成

  • 接触改質: ナフサ(石油留分)を触媒存在下で加熱・脱水素し、芳香族炭化水素を生成。
  • 熱分解: エチレン製造時の副生成物としても得られる。

8.3 アルキルベンゼンの合成

  • ベンゼンのフリーデル・クラフツアルキル化。
  • エチルベンゼン: ベンゼンとエチレンから合成(スチレン原料)。
  • クメン: ベンゼンとプロピレンから合成(フェノール原料)。

9. 重要な芳香族化合物(誘導体)

9.1 ニトロ化合物

  • ニトロベンゼン (C₆H₅NO₂): 淡黄色液体、アーモンド臭。アニリン(染料中間体)の原料。
  • トリニトロトルエン (TNT, C₆H₂CH₃(NO₂)₃): 黄色結晶、爆薬。トルエンのニトロ化で合成。

9.2 スルホン酸

  • ベンゼンスルホン酸 (C₆H₅SO₃H): 固体、水溶性。染料中間体、フェノール合成原料。
  • p-トルエンスルホン酸 (CH₃C₆H₄SO₃H): 有機合成の触媒、染料中間体。

9.3 ハロゲン化物

  • クロロベンゼン (C₆H₅Cl): 液体。フェノール、アニリン、DDTの原料。
  • ブロモベンゼン (C₆H₅Br): 液体。有機合成中間体。
  • p-ジクロロベンゼン (p-C₆H₄Cl₂): 白色結晶。防虫剤(衣類用)、消臭剤。

9.4 その他

  • ベンズアルデヒド (C₆H₅CHO): アーモンド様の香り。香料、染料原料。
  • アセトフェノン (C₆H₅COCH₃): 催眠薬(かつて)、香料、医薬品中間体。
  • ベンゾフェノン ((C₆H₅)₂C=O): 香料、紫外線吸収剤。

10. 芳香族化合物の検出・確認方法

10.1 紫外吸収スペクトル(UV)

  • ベンゼン環は230-270 nmに特徴的な吸収(π→π*遷移)を示す。

10.2 赤外吸収スペクトル(IR)

  • 芳香族C-H伸縮: 3030 cm⁻¹付近
  • 芳香環C=C伸縮: 1600, 1500 cm⁻¹付近
  • 置換様式による吸収: 900-700 cm⁻¹(面外変角振動)で置換パターンの判定が可能

10.3 核磁気共鳴スペクトル(NMR)

  • 芳香族プロトン: 6.5-8.0 ppmにシグナル
  • 置換基の影響でシフトが変化

10.4 化学的確認法

  • ニトロ化: 黄色のニトロ化合物の生成
  • スルホン化: 水溶性のスルホン酸の生成
  • ハロゲン化: ハロゲン化水素の発生

11. 芳香族化合物の応用・工業的利用

11.1 プラスチック・合成繊維原料

  • スチレン: ポリスチレン(発泡スチロール、透明プラスチック)、ABS樹脂、合成ゴム(SBR)
  • p-キシレン: テレフタル酸 → ポリエステル(PETボトル、繊維)
  • フェノール: フェノール樹脂(ベークライト)
  • アニリン: ポリウレタン原料

11.2 染料・顔料

  • アニリンから合成されるアゾ染料
  • フタロシアニン顔料(青色、緑色)
  • アリザリン(赤色染料)

11.3 医薬品・農薬

  • アスピリン(アセチルサリチル酸)
  • サルファ剤(スルファニルアミド)
  • 除草剤(2,4-Dなど)

11.4 溶剤

  • トルエン、キシレン: 塗料、インク、接着剤の溶剤

11.5 爆薬

  • TNT(トリニトロトルエン)、ピクリン酸(トリニトロフェノール)

12. 芳香族化合物の毒性と環境問題

12.1 毒性

  • ベンゼン: 発がん性(IARCグループ1、白血病の原因)。骨髄毒性。
  • トルエン: ベンゼンより毒性は低いが、吸入による神経毒性(シンナー遊び)。
  • キシレン: 吸入毒性、皮膚刺激性。
  • ニトロベンゼン: 吸入・経皮吸収で中毒(メトヘモグロビン血症)。
  • アニリン: 毒性強く、経皮吸収に注意。
  • 多環芳香族炭化水素: 発がん性を持つものがある(ベンゾ[a]ピレンなど)。

12.2 環境問題

  • 揮発性有機化合物(VOC)として大気汚染の原因。
  • 光化学スモッグの原因物質。
  • 地下水汚染(ガソリンスタンドなどからの漏洩)。
  • ベンゼンの法的規制: 労働安全衛生法、大気汚染防止法などで厳しく規制。

12.3 安全な取り扱い

  • ドラフト内で使用し、蒸気の吸入を避ける。
  • 保護手袋・保護メガネを着用。
  • 火気厳禁(引火性が高い)。
  • 適切な廃棄処理(有害廃棄物として)。

13. 重要反応のまとめ(暗記用)

  • ベンゼンのニトロ化: (13) C₆H₆ + HNO₃ → C₆H₅NO₂ + H₂O
  • ベンゼンのスルホン化: (14) C₆H₆ + H₂SO₄ → C₆H₅SO₃H + H₂O
  • ベンゼンのハロゲン化(塩素): (15) C₆H₆ + Cl₂ → C₆H₅Cl + HCl
  • フリーデル・クラフツアルキル化: (16) C₆H₆ + R-Cl → C₆H₅-R + HCl
  • フリーデル・クラフツアシル化: (17) C₆H₆ + R-COCl → C₆H₅-CO-R + HCl
  • トルエンの酸化: (18) C₆H₅CH₃ + 3[O] → C₆H₅COOH + H₂O
  • ベンゼンの水素添加: (19) C₆H₆ + 3H₂ → C₆H₁₂
  • ベンゼンの塩素付加(光): (20) C₆H₆ + 3Cl₂ → C₆H₆Cl₆

14. 確認問題

  1. ベンゼンの構造について、ケクレ構造と実際の構造の違いを説明せよ。
  2. ベンゼンのニトロ化反応の反応式を示し、濃硫酸の役割を説明せよ。
  3. トルエンを過マンガン酸カリウムで酸化すると何が得られるか。反応式を示せ。
  4. フリーデル・クラフツアルキル化反応について、ベンゼンと塩化メチルを例に反応式を示せ。
  5. ベンゼン環の置換基の配向性について、オルト・パラ配向性基とメタ配向性基をそれぞれ3つずつ挙げ、その理由を簡単に説明せよ。
  6. ベンゼンとトルエンのニトロ化で得られる生成物の違いを説明せよ。
  7. ナフタレンの性質と用途を述べよ。
  8. ベンゼンの工業的製法を2つ挙げよ。
  9. スチレンの構造式と主な用途を述べよ。
  10. 芳香族化合物の毒性について、ベンゼンを例に説明せよ。
